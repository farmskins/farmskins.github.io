How to Check a CS2 Skin Float Value

farmskins

In CS2, float value is a numerical indicator that determines a skin’s level of wear. It ranges from 0.00 to 1.00, where:

  • 0.00 means the skin is Factory New

  • 1.00 means it's Battle-Scarred

This value directly affects the appearance and market price of the item. So if you’re looking for a skin in good condition, it’s essential to know its float value. Today, we’ll show you how and where to check float in CS2 to help you choose the best skins for your collection.

Where to Check Float in CS2?

Many players wonder: where can you see CS2 float value, especially since the game doesn’t display it by default. Fortunately, there are several reliable ways to find out a skin’s float:

  • By inspecting the item and using third-party websites

  • Using browser extensions (like CSFloat Market Checker)

  • Checking float through Steam inventory and marketplace

How to Check Float in CS2: Easy Methods

We’ve put together some of the simplest ways to check float value. If you want to verify the float of a CS2 skin, these methods will help:

Through Third-Party Websites

This is the most popular and versatile option:

csfloat.com – Just paste the inspect link of the skin or your Steam ID. The site will show you:

  • the exact float value

  • the pattern or seed

  • skin rarity (especially helpful for Blue Gems and Fades)

  • estimated market price

buff.163.com – A Chinese marketplace where float values are often displayed directly. Great for analyzing prices and skin condition on the Asian market.

Using Browser Extensions

If you frequently browse skins CS2 on the Steam Market, this method is for you:

  • Install the CSFloat Market Checker extension for Chrome or Firefox.

  • It automatically adds float information directly to the item’s Steam page.

  • You can also see the pattern for Doppler knives or Fade skins.

This is a great way to avoid overpaying for items with poor float, especially if you’re trading or buying in bulk.

Through Console Commands

For more accurate checking, you can use the in-game console:

  • Enable the developer console in game settings (Settings → Game → Enable Developer Console).

  • Open your inventory and click "Inspect" on the desired item.

  • In the console, a command like this will appear:

inspect weapon_xxxxxxxxxx

  • Copy that inspect link and paste it into a site like csfloat.com.

This method is especially useful if you want to check float before selling or trading CS2 skins.

Wear Level by Float Value in CS2

Each float value corresponds to a specific wear category, which you’ll see in the game:

Wear Level | Float Value Range

Factory New | 0.00 – 0.07

Minimal Wear | 0.07 – 0.15

Field-Tested | 0.15 – 0.38

Well-Worn | 0.38 – 0.45

Battle-Scarred | 0.45 – 1.00

However, even if two skins have the same wear label (e.g. Field-Tested), their float values might differ significantly. That means they could look visually very different, which is critical when hunting for clean-looking skins.
