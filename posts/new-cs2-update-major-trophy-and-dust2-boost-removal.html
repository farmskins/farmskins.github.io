New CS2 Update: Major Trophy and Dust2 Boost Removal

farmskins

Did the developers really decide to surprise us by releasing a new update for CS2, which is almost one and a half gigabytes? It raises many questions for us. Additionally, the developers have not been stingy with new leaks. In this article, we will break down the latest CS2 update as well as some leaks.

Finally, the developers remembered that the community maps have been updated for a long time, so in today's update, we received changes on four maps: Dust2, Nuke, Assembly, and Memento.

While the changes to the community maps were expected, here’s the most remarkable change on Dust2:

The developers decided to modify the box located between the CT spawn and A site. After the update, the defending player will get to short much faster, making it harder for the attacking side to contest mid at the start of the round. An additional bonus for the CT side is the ability to quickly return to A site during an attack or retake. Dust2 was already a CT-sided map—51.3% of rounds won in major tournaments in 2024—now the balance may tilt even further.

It will be interesting to see how esports players react to these intriguing changes.

Minor changes have also appeared on other maps. On Nuke, collision issues have been fixed; on Assembly, objects have been improved and several bugs fixed; and on Memento, the A site has been reworked, visuals enhanced, and bugs fixed.

A new server has been added in London for Europe. Now, you can play CS on seven servers, with Madrid, Paris, Vienna, Warsaw, Frankfurt, Stockholm, and Helsinki already available.

And of course, for the winners of the latest major, the developers finally delighted them with a new trophy model. And this raises many questions.

The funniest part is that there are no complaints about the trophy model itself. It is well-crafted and looks quite good. There are two variants: one for the finalists and one for the winners. But if you remember, in the previous version of CS, there was a single trophy for everyone, which was updated in the game and included the list of winners for each major. This saved a lot of disk space, as the developers didn't have to duplicate the same model repeatedly.

In the new version, not only does the text with the names of the players and teams look like a sticker without effects, but we likely have the same situation as with the map maker medals. They won't need to model a new variant each time, just add new text to the trophy, which takes us back to Valve's templated development structure.

The new CS2 update also included new leaks. New data has been added to the event file. For those who are not familiar, this file includes instructions for playing special sounds when hitting a headshot. You’ve probably noticed it because it stands out and lets the player know they hit the opponent in the head with a spray.

It also includes parameters collected for the MVP. For example: "most flashes" or "most fire damage" and so on.

In the new lines added in the section titled "bullet damage," the following have been included:

  • The victim who was hit

  • The attacker who dealt the damage

  • Distance between the attacker and the victim

  • Bullet vector coordinates (X, Y, Z)

  • Number of penetrations through objects before hitting the victim

  • Whether the bullet was a no-scope

  • Whether the bullet was fired while jumping

On one hand, someone might suggest this data is collected for anti-cheat purposes. On the other hand, it seems like new data for new MVP banners because this update also included a new unused animation for those MVP banners.
